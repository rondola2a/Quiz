# Quiz Flashcard Pages

This branch was created for publishing the mobile flashcard app with GitHub Pages.

The generated app files are prepared locally in `C:\Users\hoond\Desktop\TTT`:

- `index.html`: full standalone flashcard app
- `pages-index.html`: compressed GitHub Pages wrapper

To publish manually if needed, place `pages-index.html` at repository path `index.html`, then enable GitHub Pages from this branch and `/root`.
